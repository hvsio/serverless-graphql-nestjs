# Serverless GraphQl NestJS boilerplate

## Description

Simple boilerplate for Serverless GraphQl NestJS (Schema first)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# you can use serverless to start the server
$ sls:start
```
